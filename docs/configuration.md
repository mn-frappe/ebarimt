# Configuration

## eBarimt Settings

Navigate to **eBarimt Settings** in Frappe to configure your API credentials.

### Required Fields

| Field | Description |
|-------|-------------|
| POS ID | eBarimt POS terminal ID |
| Registration Number | Company registration number |
| Environment | Production or Sandbox |

### Multi-Company Setup

eBarimt supports multi-company configurations.

1. Go to **eBarimt Entity**
2. Create a new entity for each company
3. Link the entity to the company

## Testing Connection

Use the **Test Connection** button in eBarimt Settings to verify your credentials.
